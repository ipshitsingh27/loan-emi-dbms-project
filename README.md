# Loan EMI DBMS Project

This is a mini DBMS project based on Loan EMI Collection System.

## Features
- User login system
- Database connectivity using MySQL (MariaDB)
- EMI data fetched using SQL queries
- Displays EMI amount and status

## Technologies Used
- HTML
- CSS
- PHP
- MySQL (MariaDB)

## How to Run

1. Start MySQL server
2. Run PHP server:
   php -S localhost:8000
3. Open browser:
   http://localhost:8000
